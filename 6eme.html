Microcontrôleur


En 6ème, nous perfectionnerons notre utilisation des microcontroleurs tels que Rasberry Pi ou encore les cartes Arduino. Ceux-ci vous permettront de faire énormément de choses. Comme, par exemple, faire fonctionner un robot, créer un sonar, faire fonctionner une lampe connectée à distance ou encore créer un serveur pour héberger vos sites web.

HTML / CSS / JS / PHP

Après avoir maîtrisé la programmation côté-client avec les languages (HTML, CSS, et JavaScript), vous apprendrez la programmation coté-serveur avec le language PHP. Une nouveauté viendra perfectionner nos sites web : le SQL.

SQL / Base de données

My SQL est un logiciel de gestion de base de données dans le quel vous pourrez créer des tableaux pour stocker vos données grâce au language SQL. Ce language est déjà abordé en 5ème avec Access, nous allons maintenant l'intégrer dans les sites web.

Impression 3D

En 6ème vous entamerez aussi la construction d'une imprimante 3D sur laquelle vous avancerez tout au long de l'année afin de comprendre son fonctionnement et de créer des objets, en laissant votre imagination et l'imprimante faire le travail.
