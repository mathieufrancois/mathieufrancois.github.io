Microcontrôleur

En 5ème, nous abordons l'utilisation de microcontroleurs tels que Rasberry Pi ou encore les cartes Arduino. Ceux-ci vous permettront de faire énormément de choses comme, par exemple, faire fonctionner un robot, créer un sonar, faire fonctionner une lampe connectée à distance ou encore créer un serveur pour héberger vos sites web.

Access

Après avoir abordé Excel, dans les années précédentes, vous aborderez Acess. Celui-ci est un logiciel de gestion de base de données dans lequel vous pourrez justement importer des feuilles de calcul Excel et gérer beaucoup de données à la fois.

Excel


En 5ème, nous approfondissons les notions de fonctions dans le but de mieux les intégrer dans Access.

HTML / CSS / JS

Après avoir fait connaissance avec ces 3 langages (HTML, CSS, et JavaScript), vous apprendrez à perfectionner votre maitrise de ceux-ci. Cette année, le Java Script sera beaucoup plus utilisé que les autres années (création de formulaires interactifs, ...).
